Shruti Mukhtyar

Hello! I am a web developer living in the San Francisco Bay Area. I design and build websites, create climate data visualizations and teach workshops on Leaflet and D3 at the Geospatial Innovation Facility, University of California, Berkeley. A long time ago, on a desktop far, far away, I worked as a GIS analyst. Most of all I love working at the intersection of all things spatial and the web.

PROJECTS

Cal-Adapt

Web design and development. Creating interactive visualizations with D3, Leaflet and Mapbox using data served by the public Cal-Adapt API.

Coastal Hazard Resilience Planning in California

Developed web version of the printed guide written and designed by NOAA.

Adaptation Clearinghouse

Web design and development.

VTM

Web application for the Wieslander Vegetation Type Mapping Project built using AngularJS.
